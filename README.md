 Profile Card

Testable, accessible, responsive profile card component.

## Features

- Semantic HTML5
- All required `data-testid` attributes
- Live current time in milliseconds
- Responsive design (mobile/desktop)
- WCAG AA accessible
- No dependencies

## Local Run

1. Open `index.html` in any browser
2. Or terminal: `open index.html` / `start index.html` / double-click
3. Live reload: VS Code Live Server extension

## Test Elements

```
data-testid values:
- test-profile-card
- test-user-name
- test-user-bio
- test-user-time
- test-user-avatar
- test-user-social-links
- test-user-social-twitter/github/linkedin
- test-user-hobbies
- test-user-dislikes
```
